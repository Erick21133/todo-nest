
# TODO NEST

Simple todo backend with nestjs, typescript, passport-js.




## Installation

Install dependencies with npm

```bash
  npm install
```
    
## Usage/Examples

```javascript

-- Run development server --
npm run start:dev

-- execute migration -- 
npm run db:migrate

-- Fire studio -- 
npm run db:studio

-- Generate types
npm run db:generate 

-- Apply migrations
npm run db:migrate

```
